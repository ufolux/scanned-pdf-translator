# Scanned PDFs Translator

## Pre-requisites

```
npx playwright install
brew install pkg-config cairo pango libpng jpeg giflib librsvg 
```

## Installation

```
pnpm install
```

## Usage

```
node src/translator.js <input.pdf> <output.dir>
```
